Detailed Description

Core interface for depth sensors (SOLID ISP refactoring completed).

Provides core sensor operations: lifecycle (open/start/stop), mode configuration, frame acquisition, controls, and depth compute parameters. All depth sensors implement this interface.

Interface Segregation (ISP): Hardware-specific, recording, and playback capabilities are now in separate interfaces:

  • See also
    Adsd3500HardwareInterface for ADSD3500-specific commands
  • See also
    RecordableInterface for recording capabilities
  • See also
    PlaybackInterface for playback capabilities
    Implementations should inherit only the interfaces they truly support, eliminating stub methods that return Status::UNAVAILABLE.
